# 1.6-常见时间复杂度





| 执行次数函数举例 | 阶 | 非正式术语 |
| :--- | :--- | :--- |
| 12 | O\(1\) | 常数阶 |
| 2n+3 | O\(n\) | 线性阶 |
| 3n2+2n+1 | O\(n2\) | 平方阶 |
| 5log2n+20 | O\(logn\) | 对数阶 |
| 2n+3nlog2n+19 | O\(nlogn\) | nlogn阶 |
| 6n3+2n2+3n+4 | O\(n3\) | 立方阶 |
| 2n | O\(2n\) | 指数阶 |

**注意，经常将log2n（以2为底的对数）简写成logn**

所消耗的时间从小到大

**O\(1\) &lt; O\(logn\) &lt; O\(n\) &lt; O\(nlogn\) &lt; O\(n2\) &lt; O\(n3\) &lt; O\(2n\) &lt; O\(n!\) &lt; O\(nn\)**

> O\(5\)

> O\(3n³+1\)
